Class IV

There is only one known Class IV Green Gas Giant, so only guesses can be made about what commonality other planets of this type might share. This GGG maintains the normal red base color, but the darker red bands are nearly completely replaced with green. If one looks closely, there is a darker purple-red color on the borders of the alternating cloud bands. With the different cloud patterns and colors normal Class IV gas giants have, this class can definitely be one of the more beautiful ones to visit.
